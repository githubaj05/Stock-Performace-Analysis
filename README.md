# Stock Performance Analysis
Built an interactive Power BI dashboard for Indian Oil stock performance analysis using financial KPIs such as
Open, Close, High, Low prices and Volume. Implemented time-series trend analysis with dynamic visualizations and filters.

## Tools Used
Microsoft Power BI,
Excel/CSV dataset,
Data Visualization,
Stock Market Dataset

## Main KPIs
Open Price,
Close Price,
High Price,
Low Price,
Trading Volume,

## Analysis Performed
Year-wise stock trend analysis
Quarter-wise analysis
Month-wise performance tracking
Day-wise market behavior analysis
Volume analysis
Comparative stock movement analysis

## Dashboard Features
Interactive filters/slicers
KPI cards
Trend charts
Comparative visuals
Time-based stock analysis
